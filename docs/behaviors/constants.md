You can use the next constant alias variables for characters:

- Tab `!$TAB!`
- Backspace `!$BS!`
- Carriage return `!$CR!`
- Line feed `!$LF!`
- New Line `!$CRLF!`
- Exclamation `!$EXCL!`
- Quote `!$QUOT!`
- Ampersand `!$AMP!`
- Pipie `!$BAR!`
- Great than `!$GT!`
- Lower than `!$LT!`
- Opening parenthesis `!$OPAR!`
- Closing parenthesis `!$CPAR!`
- Opening brace curly `!$OBRC!`
- Closing brace curly  `!$CBRC!`
- Asterisk `!$STAR!`
- ? `!$QUES!`
- $ `!$DOLLAR!`
- ; `!$SEMI!`
- , `!$COMMA!`
- = `!$EQ!`
- Escape `!$ESC!`
