You can use the next constant alias variables for characters:

- TAB `!$TAB!`
- BACKSPACE `!$BS!`
- CARRIAGE RETURN `!$CR!`
- LINE FEED `!$LF!`
- WINDOWS NEW LINE `!$CRLF!`
- EXCLAMATION `!$EXCL!`
- QUOTE `!$QUOT!`
- AMPERSAND `!$AMP!`
- PIPE `!$BAR!`
- GREAT THAN `!$GT!`
- LOWER THAN `!$LT!`
- OPENING PARENTHESIS `!$OPAR!`
- CLOSING PARENTHESIS `!$CPAR!`
- OPENING BRACE CURLY `!$OBRC!`
- CLOSING BRACE CURLY  `!$CBRC!`
- ASTERISK `!$STAR!`
- ? `!$QUES!`
- $ `!$DOLLAR!`
- ; `!$SEMI!`
- , `!$COMMA!`
- = `!$EQ!`
- ESCAPE: `!$ESC!`
