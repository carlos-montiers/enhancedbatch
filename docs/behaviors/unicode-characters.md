# Unicode Characters
{: .no_toc }

## Table of contents
{: .no_toc .text-delta }

1. TOC
{:toc}

---

Enhanced Batch allow you generate any Unicode character in an easy way: `%U+N%`, where N is a hexadecimal number smaller than 110000 (0x1ADB0‬).

## Example
```
echo greek psi: %U+03A8%
greek psi: Ψ
```
