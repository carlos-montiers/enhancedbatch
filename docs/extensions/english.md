## @english
Enables or disable the translation to english of the values of the next extensions:

- @monthname
- @monthshort
- @dayname
- @dayshort

## enable
```
set @english=1
```

## disable
```
set @english=0
```

## get @english
Retrieves the current configuration value

```
echo !@english!
0
```
