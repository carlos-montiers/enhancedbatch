## get @datetimems
Retrieves the current local date and time in the format: YYYY-MM-DD HH:MM:SS.milliseconds

The milliseconds have a width of 3 and uses the local decimal point.

```
echo !@datetimems!
2019-12-31 23:00:59.136
```
